# atividade-branchs

Aplicação Todo List simples para praticar Git Flow de features e PRs.

Requisitos da atividade:

- 6 funcionalidades, cada uma em uma branch `feature/<nome>`
- Pelo menos 2 commits por branch
- 3 merges diretos no `main`
- 3 pull requests no `main`

Funcionalidades propostas:

1. `add-task` — adicionar tarefa
2. `toggle-complete` — marcar/desmarcar concluída
3. `delete-task` — excluir tarefa
4. `edit-task` — editar título da tarefa
5. `persist-localstorage` — salvar/carregar no LocalStorage
6. `filter-tasks` — filtros Todas/Ativas/Concluídas


## Rodando localmente

Abra o arquivo `index.html` no navegador.
